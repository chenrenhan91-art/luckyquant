# 策略模板库

> 构建器按用户经验分级展示模板。固定模板（tier=fixed）均**对标宽基指数**，设计目标为在控制回撤的前提下**争取跑赢买入持有**——须完成 WFO/DSR 样本外检验，不保证任何区间必胜。

| 经验等级 | 可用模板 |
|----------|----------|
| 完全零基础 | 仅 `tier=fixed`：指数双动量、Regime 波段、智能定投 |
| 有投资经验无体系 | `tier=advanced`：CTA、网格、指数增强因子、ML 等 |
| 成熟体系 | 自然语言描述为主，模板仅作参考 |

## 设计原则（固定模板）

1. **单指数 ETF 标的** — 510300/SPY 等，避免新手选股噪音
2. **显式基准** — 回测必须对比指数买入持有，报告 Alpha 与 MaxDD
3. **Regime / 动量过滤** — 借鉴 Antonacci 双动量、Faber 200 日均线、Jegadeesh-Titman 12-1 动量
4. **防未来函数** — 信号 `shift(1)`，月度再平衡在次日执行
5. **诚实预期** — 部分年份可能跑输指数；优势在于熊市控回撤与风险调整收益

## 新手推荐 ⭐（固定模板）

### 1. 指数双动量趋势 (`dual_ma`)

| 属性 | 说明 |
|------|------|
| 难度 | ★☆☆ 入门 |
| 对标基准 | A股 沪深300 (000300) / 美股 SPY |
| 研究依据 | Antonacci Dual Momentum · Faber TAA · 12-1 绝对动量 |
| 核心逻辑 | 252 日收益>0 且 EMA50>EMA200 → 满仓指数 ETF；否则持短债/货币 ETF |
| 指标 | EMA(50/200), SMA(200), 12 月收益 |
| 白话 | 大势向上持有指数，大势转弱退守现金/债券，减少熊市回撤 |
| 风险建议 | 2-3 分 |
| 超额来源 | 熊市规避 + 趋势延续阶段的满仓暴露 |
| 回测必做 | 对比 SPY/510300 买入持有；含 2008/2015/2022 熊市；信号 lag 1 日 |

### 2. 指数 Regime 波段 (`rsi_bounce`)

| 属性 | 说明 |
|------|------|
| 难度 | ★☆☆ 入门 |
| 对标基准 | 沪深300 / SPY |
| 研究依据 | Regime filter (SMA200) · Connors 短期 RSI |
| 核心逻辑 | 仅当 Close>SMA(200) 时，RSI<35 拐头买入；RSI>65 或破 SMA200 卖出 |
| 指标 | RSI(14), SMA(200) |
| 白话 | 只在「长期趋势向上」时做超卖反弹，避免熊市接飞刀 |
| 风险建议 | 1-2 分 |
| 超额来源 | 牛市/震荡期的波段 Alpha；熊市空仓 |
| 回测必做 | 分牛/熊/震荡子样本报告超额收益 |

### 3. 指数智能定投 (`dca`)

| 属性 | 说明 |
|------|------|
| 难度 | ★☆☆ 入门 |
| 对标基准 | 沪深300 / SPY；对比「普通定额定投」 |
| 研究依据 | Dollar-Cost Averaging · SMA 估值倾斜 |
| 核心逻辑 | 每月定投；价<SMA200 加码 1.5×；价>SMA200×1.15 减至 0.5× |
| 指标 | SMA(200), RSI（辅助） |
| 白话 | 越跌买越多、越涨买越少，争取跑赢「傻定投」 |
| 风险建议 | 1-2 分 |
| 超额来源 | 估值倾斜带来的成本优势（长期） |
| 回测必做 | ≥5 年样本；对比 naive DCA 与 buy-and-hold |

## 模板选择决策树（零基础）

```
你的目标？
├── 大势涨跌都能活 → 指数双动量趋势 ⭐（首选）
├── 想波段但怕熊市 → 指数 Regime 波段 ⭐
├── 长期省心存钱 → 指数智能定投 ⭐
└── 还不确定 → 双动量 + 回测对比沪深300/SPY
```
