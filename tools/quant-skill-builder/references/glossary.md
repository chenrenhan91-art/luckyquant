# 量化术语白话词典

> AI 与用户沟通时必须优先使用「白话列」，专业术语作补充。

| 术语 | 白话解释 | 举例 |
|------|----------|------|
| 回测 | 用历史数据模拟「如果当时这样交易会怎样」 | 用 2020-2024 年数据测试你的买卖规则 |
| 过拟合 | 策略只是「背住了」历史答案，未来会失效 | 考卷做十遍得满分，换一套题就不行了 |
| 样本外 (OOS) | 留一段从没用来调参的数据做最终考试 | 前 4 年调参，第 5 年验证 |
| Sharpe 比率 | 每承担 1 份风险赚多少收益 | 越高越好，>1 算不错，>2 很好 |
| 最大回撤 | 从最高点到最低点最多亏多少 | 100 万最高变 120 万再跌到 90 万，回撤 25% |
| 未来函数 | 用了「当时还不知道」的信息做决策 | 用收盘价信号却在开盘价成交 |
| 滑点 | 实际成交价和预期价格的差距 | 想 10 元买，实际 10.05 元成交 |
| 仓位 | 某只股票占总资金的比例 | 10 万里买 2 万某股 = 20% 仓位 |
| 止损 | 亏到某个比例就强制卖出 | 亏 5% 就卖，防止越亏越多 |
| 杠杆 | 借钱放大交易规模 | 1 倍 = 不借钱，2 倍 = 借同样多的钱 |
| 均线 | 过去 N 天价格的平均值 | 20 日均线 = 最近 20 天收盘价的平均 |
| 金叉/死叉 | 短期线穿过长期线 | 金叉=上穿（常视为买入），死叉=下穿（卖出） |
| RSI | 衡量涨跌幅度的指标，0-100 | 低于 30 可能超卖，高于 70 可能超买 |
| MACD | 两条线的差值，看动能变化 | 线上穿信号线 = 动能转强 |
| 布林带 | 价格正常波动的上下通道 | 触下轨可能反弹，触上轨可能回落 |
| ATR | 最近价格波动有多大 | 用来设止损距离，波动大止损宽 |
| Walk-Forward | 滚动窗口反复做「训练+考试」 | 每年用过去数据调参，下一年验证 |
| DSR | 校正「试了很多策略才找到一个好看的」 | 防止「筛了 100 个才展示最好的那个」 |
| PBO | 策略是曲线拟合的概率 | 超过 50% 说明很可能是运气 |
| T+1 | A股今天买的明天才能卖 | 回测 A 股必须模拟这个规则 |
| 涨跌停 | 单日涨跌有 10% 上限（主板） | 涨停买不到，跌停卖不出 |
| 模拟盘 | 用真实行情但假钱交易 | 实盘前的最后一关 |
| 因子 | 给股票打分的某个维度 | 市盈率低 = 价值因子分高 |
